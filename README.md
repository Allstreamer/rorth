# Rorth
Rorth is a Porth-Simulator written in Rust. 
This is not meant to be a serious project, but one for me to learn Rust.

## Quick start

Build
````console
$ cargo build --release
````
Use
````console
// Windows
$ target\release\rorth test.porth
````
